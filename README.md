## 🎮 프로젝트 소개

9회말, 극적인 순간. 당신은 타자가 될 것인가, 투수가 될 것인가?

실제 사회인야구 기록을 게임에 반영한다는 아이디어에서 시작된 확률 기반 야구 시뮬레이션 게임입니다.

- 9회초 2점 리드 세이브 상황
- 사용자는 투수로 플레이를 진행
- 3아웃을 잡아서 세이브하기

- 9회말 2점 뒤진 역전 상황
- 사용자는 타자로 플레이를 진행
- 3아웃이 되기 전까지 역전하기

## 1. 게임 기본 안내

### 1.1 게임 모드

- 타자 모드와 투수 모드 2가지
- 각 모드는 독립적으로 플레이 가능

### 1.2 투수 모드

- 초기 상황 : 9회초 0아웃, 홈팀(user) 2 : 0 어웨이(com) - 2점차 세이브 상황
- 사용자는 투수로 플레이를 진행
- 목표 : 3아웃을 잡아서 세이브하기
- 승리 조건 : 리드한 상태로 3아웃
- 패배 조건 : 역전 허용 시 즉시 패배

### **1.3 타자 모드**

- 초기 상황 : 9회말 0아웃, 홈팀(user) 0 : 2 어웨이(com) - 2점차로 지고 있는 상황
- 사용자는 타자로 플레이 진행
- 목표 : 3아웃이 되기 전까지 역전하기
- 승리 조건 : 역전 시 즉시 승리
- 패배 조건 : 지고 있는 상태로 3아웃

## 2. 게임 진행 방식

### 2.1 스트라이크 존

- 9개의 구역으로 분할 (1 ~ 9번) 3x3 형태
- 볼은 없음 (스트라이크만 존재)

### 2.2 투수 모드 진행 방식 (사용자가 투수)

- 매 투구마다 컴퓨터가 Hot Zone (타자 선호 구역)과 Cold Zone (타자 비선호 구역) 랜덤 지정
- Hot Zone과 Cold Zone은 투구 이전에는 공개되지 않으며, 투구 결과 시에만 표기
- 사용자는 9개의 구역 중 1개를 선택하여 투구를 진행
- 볼 카운트는 없음
- Hot Zone의 경우 안타 확률 증가 / Cold Zone의 경우 땅볼, 플라이 아웃, 헛스윙 확률 증가
- 안타의 종류 1루타, 2루타, 3루타, 홈런의 경우 “강민준”의 게임원 타자 통산 기록을 반영

**예시**
- Hot Zone의 경우 “강민준”의 게임원 통산 타율 x 1.5 ⇒ 약 0.552
- Cold Zone의 경우 “강민준”의 게임원 통산 타율 x 0.5 ⇒ 약 0.184

### 2.3 타자 모드 진행 방식 (사용자가 타자)

- 매 투구마다 컴퓨터가 9개의 구역 중 1개의 공간으로 투구를 진행
- 사용자는 9개의 구역 중 1개를 선택하여 타격을 진행 (Hot Zone 생성) ⇒ 노림수
- Cold Zone의 경우 선택한 Hot Zone을 제외한 8개의 구역 중 1개 랜덤 생성
- 볼 카운트는 없음
- Hot Zone의 경우 안타 확률 증가 / Cold Zone의 경우 땅볼, 플라이 아웃, 헛스윙 확률 증가
- 안타의 종류 1루타, 2루타, 3루타, 홈런의 경우 “강민준”의 게임원 타자 통산 기록을 반영
- Hot Zone과 Cold Zone이 아닌 경우 타자는 안타(1루타, 2루타, 3루타, 홈런) 혹은 헛스윙만 가능

**예시**
- Hot Zone의 경우 “강민준”의 게임원 통산 타율 x 1.5 ⇒ 약 0.552
- Cold Zone의 경우 “강민준”의 게임원 통산 타율 x 0.5 ⇒ 약 0.184

### 2.4 안타 종류 결정 (강민준 게임원 기록 2025.11.09 기준)

| 안타 종류 | 개수 | 확률 |
|----------|-----:|------:|
| **1루타** | 33 | **0.541** |
| **2루타** | 23 | **0.377** |
| **3루타** | 4  | **0.065** |
| **홈런**  | 1  | **0.017** |
| **총 합**  | 1  | **1.0** |

### 2.5 병살타, 희생플라이, 진루

- 타자 강민준은 병살타를 치지 않기 때문에 존재하지 않음
- 희생플라이 조건
    - 주자가 3루에 있고, 플라이 아웃 판정인 경우
    - 총 수비 숫자 9명 중 외야수(3명)가 잡을 경우 9명의 수비수 중 외야수 3명 33.3%의 확률로 희생플라이 판정
- 땅볼 아웃의 경우 진루는 없음

### 2.6 핫존, 콜드존, 일반존 확률표 예시 (강민준 게임원 기록 2025.11.09 기준)

| 구분 | 안타 | 헛스윙 | 아웃 |
| --- | --- | --- | --- |
| 핫존 (1.5배) | 0.55 | 0.225 | 0.225 |
| 일반존 | 0.367 | 0.316 | 0.316 |
| 콜드존 (0.5배) | 0.18 | 0.41 | 0.41 |

210 타석 중에 삼진을 제외한 아웃이 59개

59개의 아웃카운트 + 4개의 희생플라이
따라서 희생플라이의 확률은

4 / 63 → 0.06

원래는 3루에 주자가 있으면서 아웃카운트가 2개 미만인 경우에서 잡힌 아웃 카운트 개수로 구해야하지만 기록이 존재하지 않아서 정확한 계산 불가.

## 3. 점수 및 주자 관리

### 3.1 주자 진루 규칙

- 타자의 경우 안타 종류 만큼 진루
- 주자의 경우 타자의 안타 종류 만큼 진루 (ex. 2루타 → 2개 베이스 진루)

### 3.2 득점 계산

- 주자가 홈까지 오면 득점
- 투수 모드 : 원정팀이 득점
- 타자 모드 : 홈팀이 득점

### 3.3 게임 종료 조건

- 투수 모드
    - 3아웃 달성 → 승리
    - 역전 허용 → 패배
    - 동점 → 무승부
- 타자 모드
    - 역전하는 즉시 → 승리
    - 동점인 상태로 3아웃 → 무승부
    - 지고있는 상태로 3아웃 → 패배
